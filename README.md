## 🚀 Описание проекта

Проект создан в рамках домашнего задания по курсу «Продвинутый JavaScript в браузере». Основная цель — реализация логики проверки карт, сборка интерфейса через Webpack и настройка автоматического CI/CD процесса с Unit и E2E тестированием.

### Реализованный функционал:
* Валидация номера карты по алгоритму Луна.
* Визуальная подсветка логотипов платёжных систем (Visa, Mastercard, Mir и др.) при вводе номера.
* Полноценное автоматическое тестирование сборки при каждом пуше.

---

## 🛠 Стек технологий

* **Язык:** JavaScript (ES6+)
* **Менеджер пакетов:** npm
* **Сборщик:** Webpack 5 (сборка стилей, картинок и HTML)
* **Тестирование:** Jest + JSDOM (Unit), Puppeteer (E2E)
* **Линтинг:** ESLint
* **CI/CD:** GitHub Actions + GitHub Pages
